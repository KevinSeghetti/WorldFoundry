Instruction Format

All instructions are 32 bits long. They come in three different formats.

I-Type (Immediate)

31     26 25     21 20     16 15                           0   
   op         rs        rt            immediate



J-Type (Jump)

31     26 25                                               0  
   op                target



R-Type (Register)

31     26 25     21 20     16 15     11 10       6 5       0  
   op         rs        rt       rd        shamt     funct

Definitions

opa 6-bit operation code
rs a 5-bit source register specifier
rt a 5-bit target (source/destination) register or branch condition
immediate a 16-bit immediate, branch displacement or address displacement
target a 26-bit jump target address
rd a 5-bit destination register specifier
shamt a 5-bit shift amount
funct a 6-bit function field

Floating Point

I-Type (Immediate)

31     26 25     21 20     16 15                           0   
  cop        base       ft            offset

R-Type (Floating Point Register)

31     26 25     21 20     16 15     11 10       6 5       0  
  cop        fmt        ft       fs         fd       funct

FP Definitions

copa 6-bit operation code
base a 5-bit base register specifier
fmt a 5-bit format specifier
fs a 5-bit source1 register
ft (R) a 5-bit source2 register;
(I) source (for stores) or destination (for loads) FPA register specifier
fd a 5-bit destination register
funct a 6-bit function field
offset a 16-bit signed immediate offset

The fmt field refers to operand size.

Format Definitions

MnemonicSizeCode
S single precision - binary floating-point 0x10
D double precision - binary floating-point 0x11
W single precision - binary fixed-point 0x14
